# ***Aplicacón de Factura en modo consola***

Ejemplo para la creación de una aplicación de consola que permita gestionar facturas.

# Requisitos

- Rust 1.70 o superior
- Cargo

# Instalación

```bash
cargo install cargo-watch
```

# Ejecución

```bash
cargo watch -x run
```

# Pruebas

```bash
cargo test
```

# Formateo

```bash
cargo fmt
```
